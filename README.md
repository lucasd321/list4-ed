# Sistema de Exercícios em Java

Projeto desenvolvido para praticar modularização em Java.

## Estrutura

- Main.java → Contém o método main e o menu do programa.
- Funcoes.java → Contém todas as funções responsáveis pelos cálculos e validações.

## Funcionalidades

1. Ranking de Notas
   - Calcula média
   - Conta alunos acima da média
   - Encontra maior nota

2. Caixa Eletrônico
   - Calcula quantidade de cédulas (50, 20 e 10)

3. Validador de Senha
   - Verifica tamanho mínimo
   - Verifica caractere especial

4. Conversão de Tempo
   - Converte segundos em horas, minutos e segundos

5. Verificação de Elementos Únicos
   - Identifica números repetidos em um vetor

## Requisitos Atendidos

- Cada funcionalidade está em uma função específica.
- Não há uso de variáveis globais.
- O main apenas organiza as chamadas das funções.

## Execução

Compilar:
javac Main.java Funcoes.java

Executar:
java Main
